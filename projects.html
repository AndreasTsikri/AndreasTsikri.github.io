Completed projects

  • Diploma Thesis - Wind turbine blade design with noise reduction goal, using metaheuristic optimization algorithm PSO

    • In order to reduce the noise of turbine multiple techniques was developed in recent years. Here we tried to reduce the SPL(dB) aero-acoustic noise of 2 prototype wind turbines by using a noise model developed by nasa coupled with metaheuristic optimization algorithm PSO. The flow characteristic was calculated by using a in-house solver (FOIL2W) and the outputs was fed into a noise model. Based on this output the optimization was took place. Finally, we replaced the previous computational-costly procedure with a Machine Learning model (deep neural network) and made some more optimizations and test PSO parameters.

  • Football match prediction using kaggle dataset

    • Using this kaggle dataset i visuallized the dataset then adter making some feature engineering trying to train some models, a simple Linear Neural Network and 2 MLP in order to predict the football match results based on bet-companies odds and football match characteristics. Finally, i implemented a clustering algorithm K-means. For more details check the pdfs below

  • Assignment in Robotics using matlab

    • I Solved the straight kinematic problem of a 6 DOF robot. Path planning was also implemented using polynomials based on path planning theory
